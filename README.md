# 🏖️ Projet Lido Serena - Système de Gestion Restaurant

Bienvenue dans le dépôt du projet **Lido Serena**. Cette application web a été conçue pour moderniser la prise de commande et la gestion du restaurant, remplaçant l'ancien système papier par une interface numérique intuitive et performante.

---

## 📝 Présentation du projet
Le système répond aux besoins de modernisation de la direction pour éviter les erreurs de service et piloter les stocks avec précision :V
* **Gestion de la Carte** : Interface administrateur pour ajouter, modifier ou supprimer des plats, menus et boissons.
* **Tableau de Bord (Dashboard)** : Visualisation de données via des graphiques (plats les plus commandés, fréquentation, CA) pour mieux comprendre les besoins clients.
* **Sécurité** : Accès restreint par authentification avec mot de passe sécurisé.

---

## 🚀 Prérequis pour lancer l'application
Pour faire fonctionner l'application en local, vous avez besoin d'un environnement type **XAMPP**, **WAMP** ou **MAMP** :

1.  **Serveur Web** : Apache.
2.  **Base de données** : MySQL.
3.  **Langage** : PHP 7.4 ou supérieur.
4.  **Navigateur** : N'importe.
